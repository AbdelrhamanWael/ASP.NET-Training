# Phase 02: Web API Basics

This folder contains the tasks for Phase 02, focusing on building robust, well-structured, and strictly validated REST APIs in ASP.NET Core.

## 🚀 Quick Start / How to Run
1. Navigate to any of the task folders (e.g., `cd task-03-products-categories-api/Products & Categories API`).
2. Run `dotnet run` to start the application.
3. Open your browser and navigate to `https://localhost:<port>/swagger` to interact with the API via Swagger UI.
