# Phase 02: Web API Basics

This folder contains the tasks for Phase 02.

## Tasks
- **Task 00: API Setup** - Initial setup and configuration for Web API projects.
- **Task 01: REST Routing Drills** - Set up the ASP.NET Core Web API project and defined controllers for various routing drills.
- **Task 02: Student Management API** - A robust ASP.NET Core Web API for managing student records.
- **Task 03: Products & Categories API** - Developed a Web API to manage products and categories.
- **Task 04: Book Store API** - Created an API for managing a bookstore.
- **Task 05: Postman & Swagger Evidence** - Included Postman collections and Swagger evidence for API testing and documentation.
- **Task 06: API Standards & Refactor Pack** - Refactored API codebase to adhere to standards, added DTOs, service layers, and improved error handling.
- **Task 07: Interview Answers** - Provided answers to conceptual API questions.
