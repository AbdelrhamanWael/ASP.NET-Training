# Task 07 - Interview Answers

This directory contains the answers to the technical interview preparation guide.

## Files Included

- `Answers.txt`: Contains detailed answers to fundamental C#, OOP, Architecture, LINQ, SQL, and Professional Practices questions.
- `Technical_Interview_Prep_Guide.pdf`: The reference guide containing the original questions.

## Topics Covered

1. **Object-Oriented Programming (OOP) in C#**: Concepts like classes vs. objects, encapsulation, and properties vs. fields.
2. **Application Architecture**: The role of service classes and separation of concerns.
3. **Collections & LINQ**: Differences between Arrays, Lists, and Dictionaries, along with LINQ operations (`Where`, `Select`, `GroupBy`, `Skip`, `Take`).
4. **Databases (SQL)**: Understanding Primary/Foreign keys, relationships, and the purpose of `JOIN`.
5. **Professional Practices**: The importance of version control (GitHub), meaningful READMEs, multiple commits, and professional delivery.
